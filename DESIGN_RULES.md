# DESIGN_RULES

## 기본 방향

- 슈퍼마리오 원본 이미지, 음악, 효과음은 사용하지 않는다.
- 밝고 귀여운 오리지널 8비트풍 그래픽과 사운드로 만든다.
- 아이들이 실패해도 부담이 적은 난이도를 우선한다.

## 난이도

- 점프 간격은 넓게 잡는다.
- 낙사 구간은 최소화한다.
- 몬스터에 닿아도 즉시 게임오버가 아니라 뒤로 튕김 + 짧은 무적 시간으로 처리한다.

## UI

- 화면 상단에는 현재 구간, 코인 수, 목표 시간을 표시한다.
- 시작 전에는 간단한 시작 안내만 보여준다.
- 모바일/태블릿에서도 테스트할 수 있도록 화면 하단에 터치 버튼을 둔다.

## 사운드

- WebAudio로 합성한 오리지널 8비트풍 사운드를 사용한다.
- 배경음, 점프, 코인, 아이템, 파이프, 충돌, 클리어 효과음을 분리한다.
- 나중에 실제 사운드 파일로 교체할 수 있도록 게임 로직과 사운드 호출을 분리한다.
