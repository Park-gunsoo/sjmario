# CURRENT_STATUS

## 현재 상태

- Phaser + Vite + TypeScript 기반 웹게임 MVP를 구현한다.
- 프로젝트 시작 시점에는 기존 코드와 지침 문서가 없었다.

## 실제 구현 데이터

- 게임 흐름: 1층 3분 -> 지하 3분 -> 1층 1분 -> 깃발 클리어
- 파이프 이동: 아래 키 입력으로 자동 진입
- 아이템 변신: 브로콜리 = 아빠, 꽃 = 엄마
- 몬스터: 거북이형, 밤송이형 오리지널 캐릭터

## 임시 리소스

- 아빠/엄마/아이 캐릭터는 `img/`의 캐릭터 시트를 잘라서 적용했다.
- 잘라낸 게임용 스프라이트는 `public/assets/characters/`에 있다.
- 그래픽 일부는 Phaser 코드로 생성한 임시 픽셀 스타일이다.
- 로컬 실행에서는 `public/assets/sound/`의 파일을 우선 사용한다.
- 공개 배포에서는 사운드 원본 파일을 올리지 않고 WebAudio 합성음을 사용한다.

## 아직 하지 않은 것

- 실제 아빠/엄마 사진 적용
- 맵 에디터
- 저장 기능, DB, API
- 배포 설정

## 캐릭터 이미지 적용

- 기본 아이 캐릭터는 서준이다.
- 시작 화면에서 서준/서진 큰 캐릭터 카드를 눌러 선택할 수 있다.
- 키보드 `1`은 서준, `2`는 서진 선택이다.
- 시작 화면의 `게임 시작` 오른쪽에 태블릿용 `전체화면` 버튼이 있다.
- 브로콜리를 먹으면 아빠 캐릭터로 바뀐다.
- 꽃을 먹으면 엄마 캐릭터로 바뀐다.
- 아빠/엄마로 변신하면 캐릭터가 2배 크기로 커진다.
- 원본 시트를 바꾸면 `python scripts/generate_character_sprites.py`를 다시 실행한다.

## 사운드 적용

- 아래 파일은 로컬 플레이용이다. 공개 GitHub Pages 배포에는 포함하지 않는다.
- 1층 배경음: `01. Ground Theme.mp3`
- 지하 배경음: `02. Underground Theme.mp3`
- 코인: `smb_coin.wav`
- 점프: `smb_jump-super.wav`
- 파이프: `smb_pipe.wav`
- 변신 아이템: `super_mario_bros_mushroom_sound_effect_58k.mp3`
- 클리어: `12. Level Clear.mp3`
- 몬스터 충돌: `09. Game Over.mp3`
